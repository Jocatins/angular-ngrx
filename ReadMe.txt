Uses of NGRX
It is a state management library for Angular Applications.

- It offers a centralized store for managing the state of an application. It makes it easier to manage
 and share state across different components and modules. and to maintain consistency throughout the application

 - It helps reduce unnecessary component rerender, thereby improving performance

 - It makes the application easier to predict and to debug.

 - It supports time-travel debugging, which allows developers to debug state changes by moving
 forward and backward through the application's state history.

 - It integrates seamlessly with angular
